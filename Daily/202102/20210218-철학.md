# 철학개론 7강

### #1. 모든 생문을 자기 보존의 노력을 계속한다. -> `목적론` [부분들은 전체를 위하여 기여한다.]
- 인간의 신체를 구성하는 기관과 세포는 자신이 해야만 하는 일만 일사분란하게 움직이고 생명을 다한다.
- 그 규칙에서 벗어나는게 `암세포`이다.
- 그렇다면 우주의 구성원인 인간은 우주의 `부분`이므로 주어진 역할에서 벗어났을 때 `암`적 존재가 되는 것일까?

### #2. `영혼` animare : `생명을 불어넣다`는 의미
- 물건은 `존재` 자체이지만, 생명체는 `생명`이 중심
- 생명은 생명을 지닌 식물, 동물이 지닌 특징으로 `삶`이 끝날 때까지 유지되는 방향성이다.

### #3. 대상론
- 전통적인 입장은 **인간은 다른 동물들과 차원(레벨)이 다르다는 것**
  - 영혼 + 육신 = 생명체, 따라서 영혼이란 육신의 `형상`이 된다.
  - 사물(무생물) : 질료 + 형상
  - 생물 : + 생장혼
  - 동물 : + 감각혼
  - 인간 : + 이성혼
  - 생각 : 인간을 다른 동물들과 다르다고 정의함으로써 고유한 인간의 아이덴티티를 탐구한다.
    - 그러나 정말 인간이 다른 동물과 다를까? 다르다는 것은 정도의 차이 일까. 있음과 없음의 차이일까?
- 형상은 `부모`에게로부터, 그러나 `이성혼`은 신이 창조한 것
  - 그러나 나를 자각하게 하는 `감각`도 중요함
- `생존조건`과 `원인`은 다른 것
### #4. `지향적 동화`란?
- 내가 인식하는 감각 대상과 마주하면, 어떤 의미에서 내가 대상이 되는 것
- 그러나 대상은 아무런 변화가 없다
### #5. `동화작용`이란?
- 무엇과 동화된다는 것은 반드시 `죽음`의 과정이 필요하다.
- 생각 : 내가 신과 동화된다는 것은 반드시 `내`가 죽여야만 한다.
  - 그러나 내가 `신`의 원료로 변화되었을 때 나는 다시 `부활`한다.
  - 이것이 `rebirth`
### #6. 개체 보존도 중요하지만 그것보다 앞선 것은 종 보존이다.
